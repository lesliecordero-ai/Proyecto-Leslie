Clonar archivos en git hub:
1. git clone (link de github)
2. Crear un archivo: touch (nombre archivo)
3. Copiar un archivo fasta en el archivo vacio con cp 
4. Ingresar al archivo con nano y luego salir
Ctrl + O → Guardar cambios
Enter → Confirmar nombre del archivo	
Ctrl + X → Salir del editor
5. Agregar l archivo al area de preparacion de github: git add (nombre de archivo creado)
6. Muestra los archivos modificados, nuevos o pendientes de subir al repositorio: git status (archivo)

8. Corre el comando "git push origin main" si sale error intentas lo siguiente:
git config --global user.name "usuario github"
git config --global user.email "correo github" 
Luego vuelve a colocar el comando git push origin main
9. Revisar si se hicieron los cambios en github
